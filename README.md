# SRANSOM

Linux Dedicated Ransomware

## Overview

SRANSOM is a ransomware that dedicated for a machine with linux as its operation system. When the script executed, it will encrypt every file on any format without exception and proceed to send the key to telegram. 
You can hide this script inside anything. 

SRANSOM first created at 09/09/2025 and finished at 04/10/2025. 

**NO AI IS USED IN THIS PROJECT**
